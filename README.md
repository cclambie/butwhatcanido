# What Could I Do? ([WhatCouldIDo.craiglambie.com](http://WhatCouldIDo.craiglambie.com))

A high-impact, single-page web application designed to educate and inspire individual climate action. This project uses the visual reality of Mount Hotham's changing landscape to ground the global "1.5°C" goal in a local, tangible context.

## 🏔️ The Concept

"The difference between snow and rain is less than 1.5°"

The project centers on a hard-hitting visual of Mount Hotham's Snake Gully. It demonstrates how a tiny shift in global temperature translates to the success or failure of an entire alpine ecosystem and industry.
Creative Commons assets that people can use and share, feel free to contribute to this.

## 🎯 Purpose

This page breaks down climate action into manageable, tiered steps across five key areas of life:

* **Food:** Shifting from high-emissions beef and dairy to plant-based alternatives.

* **Transport:** Replacing car trips with bikes, public transport, and e-mobility.

* **Holidays:** Reducing the massive impact of aviation through slow travel and local exploration.

* **In Your Job:** Leveraging workplace influence for systemic change.

* **Socials:** Using personal influence and community organizing to shift social norms.

## 🛠️ Tech Stack & Deployment

* **Language:** HTML5 / CSS3 (Modern Responsive Design)

* **Typography:** Inter (via Google Fonts)

* **Deployment:** GitHub Pages

* **Domain:** [WhatCouldIDo.craiglambie.com](http://WhatCouldIDo.craiglambie.com)

## 📁 Repository Structure

* `index.html`: The complete single-page application containing all styles and content.

* `CNAME`: Configuration for the custom domain.

* `/assets`: Folder for local image hosting if not using CDN links.

## 🚀 How to Use / Edit

To make changes to the actions or emissions data:

1. Clone the repository.

2. Edit the `index.html` file, create other pages, create ideas. 
Each action is contained within an `.action-card` div marked as `easy`, `medium`, or `hard`.

3. Commit and push to the `main` branch to trigger a GitHub Pages redeploy.
